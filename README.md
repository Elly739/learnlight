# LearnLight

LearnLight is an offline-first microlearning web app designed to help students access ICT lessons and quizzes with limited or no internet connectivity.

##  SDG Alignment
**SDG 4 – Quality Education**  
LearnLight improves access to quality learning materials in low-connectivity environments.

##  MVP Features
- Offline access to ICT lessons
- Downloadable lesson packs
- Short quizzes for knowledge checks
- Progress stored locally on the device

##  Sample ICT Topics
- Introduction to ICT
- Computer Hardware vs Software
- Internet Basics and Online Safety

##  Planned Tech Stack
- React + Vite
- Progressive Web App (PWA)
- IndexedDB (local storage)
- Supabase (sync & backend – later)
